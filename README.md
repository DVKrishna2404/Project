Product Performance Analysis

The purpose of this project is to analyze product performance using sales data to identify key trends, top-performing products, and opportunities for improvement. The analysis was carried out using Microsoft Excel and Power BI. Excel was used for initial data cleaning and preparation, while Power BI was used for interactive data visualization and insights generation. The final report provides clear visual representations of product sales, revenue, and performance patterns to support data-driven business decisions.


Tools and Technologies Used:
•	Microsoft Excel: For data cleaning, formatting, and basic analysis (formulas, pivot tables, charts).
•	Microsoft Power BI: For building interactive dashboards and visualizations.
